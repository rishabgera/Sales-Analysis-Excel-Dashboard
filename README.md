# Excel-Dashboard
# 🛍️ Vrinda Store Sales Dashboard (Excel Project)

## 📌 Overview
This project is an Excel-based sales analysis of an e-commerce store (Vrinda Store). The dashboard provides insights into customer behavior, sales trends, and platform performance.

---

## 🎯 Objectives
- Analyse sales performance across states
- Understand customer demographics
- Identify top-performing sales channels
- Track order status trends

---

## 📊 Dashboard Preview
([Dashboard] dashboard.png)

---

## 📂 Dataset
- Source: Vrinda Store Sales Data
- Tool Used: Microsoft Excel
- Data includes:
  - Orders, Customers, Sales, Channels
  - Gender, Age Group, Location

---

## 🧹 Data Cleaning (Excel)
- Removed duplicates
- Handled missing values
- Standardized categories
- Converted date format

---

## 📈 Key Insights

- 👩 Women contribute more to sales than men  
- 🌍 Top 5 states drive majority revenue  
- 🛒 Amazon & Myntra are leading channels  
- 📦 Most orders are successfully delivered  
- 🎯 Adult age group is the highest contributor

Built an interactive Excel dashboard to help businesses:
